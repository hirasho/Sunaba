
MacSunaba  2017/07/15 版


○ 現在動作する機能

  + Finder からのテキストファイルのドラッグ＆ドロップによるプログラムの実行
  + 書籍本文で紹介されているプログラムの実行
  + 再起動ボタン
  + 標準のテキストテキストエディットで作成したテキストファイルを読み込んで実行
    (但し、メニューのフォーマットから "標準テキストにする" を設定することが必要)
  + キーボードの押下情報およびマウスの情報取得


○ 既知の不具合
  - キー入力が取れなくなる。
    ログ文字列をマウスで操作した後に発生しやすい。
    解決方法は MacSunaba を終了して再度起動してください。

  - グラフィックス描画している部分について
    Windows版と異なり、拡大縮小が3段階となっています。
    ログ文字表示Viewで、グラフィックスのサイズに合わせたリサイズを
    正常に実装することができませんでした。
    (たまにテキストが描画されない状態や、指定サイズにならないことがあった.)

○ 2012/02/12版との差分
  - 実行速度が向上しました
  - addという名前の部分プログラムを作るとエラーになる問題を解決しました
  - 改行コードがcrの場合に読めない問題を修正しました。
  - 不定期に55000による同期を行った時に急激に重くなる問題を修正しました
  - 拡張子がないファイルを読めない不具合を修正しました

○ 11/21版との差分
  - Sunabaプログラムにおける字句解析部分を修正しました。
